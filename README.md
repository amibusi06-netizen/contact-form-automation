# Contact Form Automation (No-Code)

## 🇫🇷 Résumé

Ce projet présente une automatisation simple permettant de gérer des demandes de contact automatiquement.
Lorsqu’un utilisateur remplit un formulaire, ses données sont enregistrées et un email de confirmation est envoyé instantanément.

---

## 🇬🇧 Project Overview

This project demonstrates a simple no-code automation system to handle contact requests automatically.
When a user submits a form, the data is stored and a confirmation email is sent instantly.

---

## Tools Used

* Google Forms
* Google Sheets
* Zapier
* Gmail

---

## Workflow

### 🇫🇷 Fonctionnement

1. L'utilisateur remplit le formulaire
2. Les données sont enregistrées dans Google Sheets
3. Zapier détecte une nouvelle réponse
4. Un email automatique est envoyé

### 🇬🇧 How it works

1. User submits the form
2. Data is stored in Google Sheets
3. Zapier detects a new row
4. An automatic email is sent
